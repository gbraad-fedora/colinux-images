coLinux image generation
========================

!["Prompt"](https://raw.githubusercontent.com/gbraad/assets/gh-pages/icons/prompt-icon-64.png)


Several scripts for creation of coLinux images:

  * CentOS 5.2, 5.3
  * Fedora 11
  * OpenSUSE 11.1
